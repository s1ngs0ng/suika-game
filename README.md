# 🍉 통이의 수박게임

> 수박을 향해 과일을 합쳐가는 물리 기반 퍼즐 게임

---

## 소개

같은 과일끼리 합치면 더 큰 과일로 진화합니다.  
체리에서 시작해 수박까지 — 얼마나 높은 점수를 낼 수 있을까요?

## 게임 방법

- 마우스를 움직여 **투하 위치**를 조준
- **클릭**으로 과일 떨어뜨리기
- 같은 과일 둘이 닿으면 **합체** → 다음 단계 과일로 진화
- 과일이 위험선을 넘으면 **게임 오버**

## 과일 진화 순서

| 단계 | 과일 | 점수 |
|------|------|------|
| 1 | 체리 | 1 |
| 2 | 딸기 | 3 |
| 3 | 포도 | 6 |
| 4 | 레몬 | 10 |
| 5 | 귤 | 15 |
| 6 | 사과 | 21 |
| 7 | 배 | 28 |
| 8 | 복숭아 | 36 |
| 9 | 파인애플 | 45 |
| 10 | 멜론 | 55 |
| 11 | 🍉 수박 | 66 |

## 기술 스택

- **Matter.js** — 2D 물리 엔진
- **Canvas API** — 픽셀아트 배경 렌더링
- **Web Audio API** — 효과음 합성
- **PokeAPI** — 배경 포켓몬 스프라이트
- Vanilla JS / HTML / CSS — 프레임워크 없음

## 특징

- 레트로 픽셀아트 다크 테마
- 달과 별이 가득한 우주 배경
- 배경에서 유영하는 랜덤 포켓몬 10마리
- 몬스터볼 클릭 시 새 포켓몬 소환 (효과음 포함)
- 우측 패널에 **진화의 고리** 표시
- 최고 기록 로컬 저장
- **PWA 지원** — 오프라인 플레이 가능, 홈 화면 설치

## 설치 (PWA)

| 기기 | 방법 |
|------|------|
| iOS | Safari → 공유 → 홈 화면에 추가 |
| Android | Chrome → 메뉴 → 앱 설치 |
